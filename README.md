## How to run JAR with Gradle

1. Create shared library jar
```bash
gradle shared:jar
```

2. To run server app jar
```bash
gradle server:jarrun
```

3. To run client app jar
```bash
gradle client:jarrun
```

## Project Structure
- Client App `client/src/java`
- Server App `server/src/java`
